なぜ専用の実行環境が必要だったか

私はWEBエンジニアで、仮想通貨の自動売買プログラムを自作して動かしている。このプログラム、一度起動したら24時間365日、止めずに動かし続けることが前提だ。

ところが普段使いのメインPCで動かすと、いくつか困ったことが起きる。

  • 動画編集や開発作業など、重い処理をすると影響が出る
  • 再起動するたびにプログラムが止まってしまう
  • 「そのソフト止めていい？」という場面で止められない心理的プレッシャー

要するに、専用の実行環境が1台欲しかったのだ。

クラウド vs 自宅PCのコスパ比較

最初に考えたのはクラウドサービスだ。AWSのEC2でもさくらのVPSでも、仮想サーバーを借りれば専用環境はすぐに手に入る。

だが、試算してみると月額費用がじわじわと気になり始めた。「年間で換算したらそれなりの金額になる……」と思い直し、別の手を考えた。

方法 初期費用 月額ランニングコスト 手間 安定性 拡張性
クラウド（VPS等） なし 数百〜数千円/月 ◎ほぼ不要 ◎高い ◎課金で即拡張
自宅の余りノートPC今回の選択 メモリ増設代のみ ◎ 電気代のみ（数百円/月） △ 自分でセットアップ △ 自宅回線依存 △ 物理的な上限あり
新しく安いPCを購入 1〜3万円程度 ◎ 電気代のみ △ 初期セットアップ △ 自宅回線依存 △ スペック固定

※クラウドの月額は構成により大きく異なります。自宅PCの電気代は常時稼働のノートPC（30〜50W）を目安に算出。

既にPCが手元にある場合、ランニングコストは電気代のみ。クラウドと比較すると、長期で見たときのコスト差は歴然だ。

実際にやったこと：古いPCをサーバー化した手順

押入れを漁ったら、7年前に買ったWindowsノートPCが出てきた。CPU は Ryzen 3、当時は普通に使えていたスペックだ。

問題はメモリが4GBしかなかったこと

さすがにメモリ4GBでは心許ない。複数のプロセスを同時に動かすには、もう少し余裕が欲しかった。そこで8GBのメモリモジュールを別途購入して増設した。

セットアップの流れ

  1. メモリを増設（8GBモジュール追加 → 合計12GB）
  2. WindowsのスタートアップにPythonスクリプトを登録
  3. ACアダプタを常時接続して、蓋を閉じてもスリープしない設定に変更
  4. リモートデスクトップを有効化して、メインPCから操作できるように設定

💡 ポイント

ノートPCはACアダプタで給電しながら「蓋を閉じてもスリープしない」設定にするのが肝心。これだけで棚やデスクの隅にコンパクトに置いておける常時稼働マシンが完成する。

12GBメモリの余力でできること

自動売買プログラムを動かしてもメモリには余裕がある。このリソースを活かして、もう1〜2個の新規プロジェクトを追加しようと考えている。

  • 📈
    仮想通貨の自動売買（複数銘柄対応） 現在は1銘柄のみだが、プロセスを増やして複数通貨ペアを同時に監視・取引することも可能。
  • 🔄
    定期実行スクリプト（データ収集・バックアップ） cronやタスクスケジューラで定期実行するデータ収集・通知・バックアップ系の処理。
  • 🌐
    ローカル開発サーバー / API サーバー 自宅LAN内の開発用サーバーとして使えば、どのデバイスからでもアクセスできる。
  • 🤖
    Bot やスクレイピングの常時稼働 SNS Bot、価格監視スクレイパー、通知システムなどをバックグラウンドで動かす。

メリット・注意点

✅ 自宅PCサーバーのメリット

  • クラウドと違い、ランニングコストが電気代だけ
  • 余っているPCがあれば実質ゼロ円で始められる
  • 物理的に手元にあるので、何か問題があれば即対応できる
  • スペックに余裕があれば複数のプロジェクトを並行実行できる
  • 中古・格安の新品PCでも十分に機能する

△ 注意点

  • 自宅のインターネット回線が落ちるとプログラムも止まる
  • 常時稼働のため電気代と発熱への配慮が必要（ノートPCはACアダプタ給電推奨）
  • クラウドと違い、スペック不足になっても即時拡張はできない
  • 古いPCは内部の埃清掃をしてから長期運用に備えるのが安心

よくある質問

Q自宅のノートPCをサーバー代わりに使うときの注意点は？
A常時起動のため電気代と発熱に注意が必要です。ACアダプタで給電し、蓋を閉じてもスリープしない設定にすることと、定期的に内部の埃を清掃することで長期運用が安定します。また自宅のインターネット回線が止まるとプログラムも止まるため、回線の安定性を事前に確認しておくと安心です。
Qメモリは何GB以上あればサーバー用途に使えますか？
A軽いプログラムを1〜2個動かす程度であれば8GBで十分な場合が多いです。複数のプロセスを並行して動かしたいなら12GB以上あると余裕が生まれます。今回紹介している新品ノートPCは最初から12GB搭載なので増設不要です。
Qクラウドサービスと自宅PC、どちらが安いですか？
A既にPCがある場合は自宅PCが圧倒的に安く、ランニングコストは電気代のみです。新しくPCを購入する場合でも、1〜2年で初期費用をクラウドの月額費用が上回るケースがほとんどです。「ずっと動かし続ける」用途であれば、自宅PCのほうが長期コストは安くなります。
Q古いPCでも24時間稼働に耐えられますか？
ACPUやメモリのスペックが最低限あれば問題ありません。ただし冷却ファンが詰まっていたり、バッテリーが劣化していると過熱・シャットダウンのリスクがあります。ACアダプタで常時給電し、バッテリーに頼らない運用にするのがポイントです。

まとめ

「24時間プログラムを動かし続けたいけど、クラウドは料金が気になる」という問題の答えは、意外と身近なところにあった。

家に眠っている古いノートPCにメモリを足してあげるだけで、月額ゼロ円の専用実行環境が完成する。しかもリソースに余裕があれば、自動売買以外の用途にも使い回せる。

  • 24時間プログラムを動かすには専用マシンがあると便利
  • クラウドは手軽だが、長期運用では自宅PCのほうがコストが安い
  • メモリ12GBあれば複数プロセスを並行実行しても余裕
  • 余りPCがない場合は安い新品 or 中古ノートPCを1台買うのがおすすめ
  • 中古でも十分。保証付きの中古なら安心して使い始められる
